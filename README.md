# DEMON MONK GAME

## demon-monk game (otherwise known as missionary-cannibal) game made in raylib.
# main files are in the demonmonk folder

The game is made entirely in raylib using C/C++. 

Tech used - 
 
C and C++ language.
Raylib - an open source library for C used for graphics rendering using OpenGL.
Blender - for textures and assets.
Godot - game engine using a custom language GDScript (used for prototyping)
LibreSprite - open source version of AESprite, a sprite drawing software

## NO GENERATIVE AI WAS USED IN CREATING ANY ARTWORK
